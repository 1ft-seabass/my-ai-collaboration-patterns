## 💡 例の見方

各例には以下が含まれます：
- **シナリオ**: どんな状況での使用例か
- **セットアップ**: パターンの導入手順
- **実行例**: 実際のコマンドやAIへの指示
- **成果物**: 何が作られるか
- **ポイント**: 注意点やコツ
